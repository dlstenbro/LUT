# LUT
An implementation of a lookup table ("LUT") using a Linked List in C

## Installation

Use git clone to get the source

```bash
git clone https://github.com/dlstenbro/LUT.git 
```

## Build & Usage

```bash
cd LUT
mkdir build && cd build
cmake ..
./lut_example
```
